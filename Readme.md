# Cybersecurity Study Notes

Personal study notes compiled by **Yadav Ghorasainee** — IS Auditor and 
Cybersecurity professional based in Perth, WA, Australia.

Covers Australian government cybersecurity frameworks, privacy law, and 
international data protection standards relevant to GRC, IS audit, and 
compliance roles in Australia.
